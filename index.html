Download TwitchVodLoader.jar

It's a small tool to download past broadcasts from Twitch.tv. Just enter the channel name in the channel textfield and start to download the videos you want. The download folder can be changed in the settings menu.

HowTo use it on Windows:

  1. Get the compiled version (jar File)
  2. Get ffmpeg https://www.ffmpeg.org/download.html
  3. place both the jar and ffmpeg in the same directory
  4. Start the tool
  5. open settings if you want to change the quality of VODs or the destination Directory ...

Use it.

MacOS and Linux

  1. Get the compiled version (jar File)
  2. Install ffmpeg on your system. It need to be there to be able to convert videos.
  3. Start the tool
  4. open settings if you want to change the quality of VODs or the destination Directory ...

Make shure Java is installed.

Features

  • Create mp4 Videos using FFMPEG
  • A m3u-playlist is created to make watching the past broadcast easier.
  • Support for the new TwitchVOD System, that stores the VOD in 3 second chunks.
  • Downloading Multiple Videos (Queue)
  • queue for converting.
